# Video intro

Copia qui il video originale della porta che si apre (dal sito attuale) con questo nome esatto:

```
assets/video/intro-porta.mp4
```

Il sito lo rileva da solo al caricamento:

- **se il file c'è** → l'intro usa il video reale (con pulsante "Salta intro");
- **se il file manca** → l'intro mostra la porta 3D animata in Three.js (stesso pulsante Salta).

Consigli: MP4 (H.264), max ~4-5 MB, risoluzione 1280×720 o 1920×1080, senza audio (il video parte in `muted`).
